# 簡易大富翁遊戲

> 供逢甲大學文化服務隊 - 寒假服務隊 原來台馬文化還能醬Young! 使用

一個使用 Node.js 和 Express.js 開發的網頁版簡易大富翁遊戲，支援多人同時獨立遊玩。

## 快速開始

### 安裝

```bash
npm install
```

### 啟動伺服器

```bash
npm start
```

伺服器將在 `http://localhost:3000` 運行

## 遊戲規則

1. **遊戲開始**：所有棋子從起點（格子 0）開始
2. **擲骰子**：每次擲一顆骰子（1-6 點），根據點數前進
3. **循環棋盤**：16 格棋盤，走到最後一格後會回到起點繼續
4. **特殊格子**：
   - 格子 4, 12（休息一次）：下一回合將跳過，無法擲骰子
   - 格子 8（再骰一次）：必須再擲一次骰子才能結束回合
5. **蓋房子**：
   - 走到任何格子（除起點和休息格）都可蓋房
   - 每回合限蓋一座房子
   - 每格可蓋無限房子，以數字標記顯示
6. **輪流進行**：玩家輪流擲骰子和移動

## 技術架構

### 後端
- **Node.js**：JavaScript 執行環境
- **Express.js**：Web 應用框架
- **UUID**：會話 ID 生成
- **會話管理**：Map 結構儲存多個獨立遊戲狀態
- **自動清理**：定期清理 24 小時未活動的會話

### 前端
- **HTML5**：頁面結構
- **CSS3**：響應式設計、動畫效果、漸層背景
- **Vanilla JavaScript**：遊戲邏輯與互動
- **會話生命週期**：每次開啟頁面都建立新會話，關閉頁面即清除

## 系統需求

- Node.js 24 或更高版本
- 網頁瀏覽器
