# 🎲 簡易大富翁遊戲

一個使用 Node.js 和 Express.js 開發的網頁版簡易大富翁遊戲。

## 🎯 遊戲特色

- **可設定棋子數量**：遊戲開始前可選擇 1-6 個棋子
- **雙骰子系統**：每次擲出兩顆骰子，根據總點數移動
- **棋盤移動**：24 格棋盤，循環移動
- **記錄系統**：追蹤每個棋子走過的格子
- **蓋房子功能**：當棋子再次走到自己走過的格子時可以蓋房子
- **即時狀態顯示**：即時顯示所有玩家的位置、走過格子數、房子數量

## 🚀 快速開始

### 安裝

```bash
npm install
```

### 啟動伺服器

```bash
npm start
```

伺服器將在 `http://localhost:3000` 運行

### 遊戲開始

1. 在瀏覽器中開啟 `http://localhost:3000`
2. 設定棋子數量（1-6 個）
3. 點擊「開始遊戲」按鈕
4. 輪流點擊「🎲 擲骰子」按鈕進行遊戲
5. 當走到自己走過的格子時，可以選擇蓋房子
6. 點擊「下一位玩家」切換玩家

## 🎮 遊戲規則

1. **遊戲開始**：所有棋子從起點（格子 0）開始
2. **擲骰子**：每次擲兩顆骰子，根據點數總和前進
3. **循環棋盤**：走到最後一格後會回到起點繼續
4. **蓋房子**：當棋子走到自己之前走過的格子時，可以選擇蓋房子
5. **輪流進行**：玩家輪流擲骰子和移動

## 🛠️ 技術架構

### 後端
- **Node.js**：JavaScript 執行環境
- **Express.js**：Web 應用框架
- **RESTful API**：遊戲狀態管理

### 前端
- **HTML5**：頁面結構
- **CSS3**：響應式設計與動畫效果
- **Vanilla JavaScript**：遊戲邏輯與互動

## 📡 API 端點

- `POST /api/init-game`：初始化遊戲，設定棋子數量
- `POST /api/roll-dice`：擲骰子並移動棋子
- `POST /api/build-house`：在目前格子蓋房子
- `POST /api/next-player`：切換到下一位玩家
- `GET /api/game-state`：取得目前遊戲狀態

## 🎨 視覺設計
